## Map
```
ccv "pat" # ccn N # s "midiN"
```

| ccn | Voice Tab |
|-----|------------|
| 10  | Gain       |
| 1-4 | Left Sidebar |
| 5   | Pitch Wheel |
| 6   | Mod Wheel  |
